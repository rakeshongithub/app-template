App Template

HTML template including bootstrap, sass, gulp tasks with browsersync. Which helps you to auto refresh your browser after every change

How to Install

npm install

How to Start app

gulp start

Once the gulp start run, you can access it on a browser by going to the web page http://localhost:3010

Gulp Start will take care of sass to css convertion with compressed version and auto refresh the browser on every change in html and css file.

JShint

gulp lint

Compress CSS/JS

gulp compress

it will create *.min file in the same directory
